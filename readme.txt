Add Google Experiments on WooCommerce pages, products and endpoints

== Description ==

WooSplit - Google Experiments for WooCommerce enables the management of Google Experiments on WooCommerce pages.  You can add as many Google Experiments as needed and specify which page, product or endpoint your experiment should be run on.

Plugin Features:

*   Add a Google Experiment and enter its "Experiment ID" and "Experiment Key" to link it to a WooCommerce page, product or endpoint.  The correct Google Experiment tracking code will then be placed on that page, product or endpoint.
*   Edit a given Google Experiment
*   Delete a give Google Experiment, individually or by bulk

For more help with Google Experiments plugin and [Conversion Rate Optimization](http://woosplit.com/) contact [WooSplit](http://woosplit.com/).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->Google Experiments screen to manage Google Experiments


== Frequently Asked Questions ==

= What happens if I have two Experiments on the same page? =

Only the first one will be added to the page.
